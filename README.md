# jusotscope

Unified offensive security toolkit, recon, scanning, AD, reporting.

Build and maintained by [Jusot](https://github.com/jusot99).

## Quick Start

```bash
pip install jusotscope

jusotscope recon example.com
jusotscope recon example.com --brute --json
jusotscope recon 8.8.8.8
```

Or download a binary from [releases](https://github.com/jusot99/jusotscope/releases).

## Tools

| Command | Description |
|---------|-------------|
| `recon` | DNS recon, subdomain enumeration, security checks |
